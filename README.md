# 📚 Pereplet (Переплёт)

Поиск книг по базе данных **Open Library** с удобным интерфейсом на базе **Nuxt 4**. Проект создан с использованием собственного инструмента для быстрой разработки — [create-nuxt-launchpad](https://www.npmjs.com/package/create-nuxt-launchpad).

![Nuxt](https://img.shields.io/badge/Nuxt-00DC82?style=for-the-badge&logo=nuxt.js&logoColor=white)
![Vue](https://img.shields.io/badge/Vue-4FC08D?style=for-the-badge&logo=vue.js&logoColor=white)
![Open Library](https://img.shields.io/badge/OpenLibrary-FF6B00?style=for-the-badge&logo=openlibrary&logoColor=white)

---

## 🌟 Особенности

- 🔍 **Поиск книг** по названию, автору или категории через Open Library API
- 📖 **Информация о книге**: обложка, описание, год издания, издательство
- ⚡ **Быстрая загрузка** благодаря SSR и оптимизации Nuxt
- 📱 **Адаптивный дизайн** для всех устройств
- 🚀 **Создано через create-nuxt-launchpad**

## 📦 Установка

### Клонирование репозитория

```bash
git clone https://github.com/KhatmullinBulat/Pereplet.git
cd Pereplet
```

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```
